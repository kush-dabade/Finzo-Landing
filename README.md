Finzo ‚Äî Fintech Landing Page
============================

Finzo is a modern fintech landing page focused on **clarity, restraint, and trust**.The goal of this project was not to maximize features, but to design a calm, intentional financial experience that communicates value without overwhelming the user.

This is a **frontend-only project**, built as a product-style landing page rather than a full SaaS.

‚ú® Highlights
------------

*   Clean, minimal visual design inspired by modern fintech products
    
*   Clear section hierarchy: value ‚Üí features ‚Üí pricing
    
*   Subtle motion and effects used intentionally
    
*   WebGL-based animated background for visual depth
    
*   Responsive layout across screen sizes
    
*   Simple, honest pricing presentation
